# CI/CD Pipeline using Jenkins + AWS + Docker

## Overview
This project demonstrates a complete CI/CD pipeline where code pushed to GitHub is automatically built and deployed on AWS EC2 using Jenkins and Docker.

---
## Tools Used
- AWS EC2
- Jenkins
- Docker
- GitHub
- Linux

---

## Workflow
1. Developer pushes code to GitHub
2. Jenkins pulls code
3. Docker image is built
4. Container is deployed on EC2
